# 커리어블룸 (Career Bloom)

> 흩어진 당신의 가능성을 모아, 가장 '나다운' 길을 설계합니다.

진로, 라이프, 커리어 코칭 서비스 웹사이트입니다.

## 프로젝트 구조

```
careerbloom/
└── frontend/     # React 프론트엔드
```

## 시작하기

### 요구사항
- Node.js 18.x 이상

### 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

개발 서버: http://localhost:5173

## 기술 스택

### Frontend
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## 페이지

| 경로 | 설명 |
|------|------|
| `/` | 메인 페이지 |
| `/about` | 코치 소개 |
| `/programs` | 프로그램 안내 |
