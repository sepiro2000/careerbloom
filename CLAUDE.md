# CLAUDE.md - 커리어블룸 프로젝트

## 프로젝트 개요
커리어블룸(Career Bloom)은 진로/라이프/커리어 코칭 서비스 웹사이트입니다.

## 디렉터리 구조
```
careerbloom/
└── frontend/          # React 프론트엔드 앱
    ├── src/
    │   ├── assets/    # 이미지 파일
    │   ├── components/# 재사용 컴포넌트
    │   └── pages/     # 페이지 컴포넌트
    ├── CLAUDE.md      # 프론트엔드 상세 컨텍스트
    └── README.md      # 프론트엔드 문서
```

## 기술 스택
- **Frontend**: React + Vite + Tailwind CSS + Framer Motion

## 개발 서버 실행
```bash
cd frontend
npm install
npm run dev
```

## 주요 페이지
- `/` - 메인 (Home)
- `/about` - 코치 소개
- `/programs` - 프로그램 안내
